## 처음 내 코드
```js
function solution(a, b) {
    var answer = 0;
    for(var i=Math.min(a,b);i<=Math.max(a,b);i++){
        answer += i;
    }
    return answer;
}
```
## 다른 사람의 코드
*가우스 공식 사용

function solution(a, b) {
    return (Math.abs(a-b)+1)*(a+b)/2;
}
